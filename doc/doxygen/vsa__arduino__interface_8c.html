void vsa_arduino_interface_write ( ArduinoInterface *  AI,
double *  u 
)

Command new motor positions u.

Parameters
AIarduino interface.
[in]u[DIMU]array of desired motor positions (rad).

Commands are clipped to within U_LLIM_RAD_SERVOx < u[x] < U_ULIM_RAD_SERVOx, and converted to units of .5 usec, before being sent to the Arduino.

Note
This function will NOT block, and there is no guarantee about when the new positions will be sent to the servos.

References ArduinoInterface::command_buffer, ArduinoInterface::port, and serialWrite().
